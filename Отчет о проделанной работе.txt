В результате слияния двух фрагментов кода был создан единый алгоритм для 
работы с двумерным массивом. Первый фрагмент подсчитывает количество строк, 
содержащих хотя бы один ноль, а второй — суммирует отрицательные значения всех
 положительных чисел в массиве. Эти задачи были объединены в один цикл, где каждый 
элемент массива проверяется на наличие нуля и положительное значение. Таким образом, 
программа одновременно решает обе задачи для одного массива. 

Исходный код для подсчета количества строк с нулем:
#колво строк с 0

 massiv =[[1,0,0],
    [6,9,5],
    [4,7,0]]
result = 0
flag = 0
for i in range(len(massiv)):
    for j in range(len(massiv[i])):
        if massiv[i][j]==0:
            result = result+1
            break
print("Количество нулей в массиве = ", result)

Исходный код для подсчета суммы положительных чисел в двумерном массиве:
massiv = [5,1,-5,-7,0]
flag = 0
for i in range(len(massiv)):
    if massiv[i]>0:
        flag =flag+massiv[i]
print(flag)

Результат объединения:
massiv = [[1, 0, 0],
           [1, -7, 1],
           [1, 0, 0]]
result = 0

# Первый фрагмент: подсчет строк, содержащих 0
for i in range(len(massiv)):
    for j in range(len(massiv[i])):
        if massiv[i][j] == 0:
            result += 1
            break

print("Количество строк, содержащих хотя бы один 0: ", result)

# Второй фрагмент: работа с одномерным массивом
flag = 0
for i in range(len(massiv)):
    for j in range(len(massiv[i])):
        if massiv[i][j] > 0:
            flag += massiv[i][j]

print("Результат обработки одномерного массива: ", flag)
